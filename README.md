# Shades-by-Jay UI Library

A modern, customizable UI component library for web projects. Includes a variety of ready-to-use components, styles, and utilities to accelerate your development workflow.

## Features
- 3D Cards
- Retro Components
- Switches
- Loaders
- Keyboard UI Elements
- Notification System
- Templates
- Playground for live testing
- Supabase integration

## Getting Started

1. **Clone the repository:**
   ```sh
   git clone <repo-url>
   ```
2. **Open the project folder:**
   Navigate to the project directory in your editor.
3. **Explore the playground:**
   Open `playground.html` to see and test all components.

## Folder Structure
```
assets/           # Images and media
css/              # Stylesheets for components and pages
js/               # JavaScript files for components and logic
admin/            # Admin dashboard
index.html        # Main entry point
playground.html   # Component showcase
```

## Usage
Include the desired CSS and JS files in your HTML:
```html
<link rel="stylesheet" href="css/style.css">
<script src="js/app.js"></script>
```
